<!-- GETTING STARTED -->

## Getting Started

After copying the code from the project, open terminal and navigate to the project folder using "cd <project/folder>".
Run the "npm i" command to install the necessary packages.
Start the server with the following command: "npm start", you should receive the "LOG [NestApplication] Nest application successfully started +1ms" message if all goes right.
Opening your browser on localhost:3000 should also grant the "hello world" message to be visible.

### Prerequisites

You must have Node.js installed to run the server.
All info and instructions in https://nestjs.com/.

### Structure

#### Main directory should contain the following folders/files:

\* Folders:

- ## src:
  - auth
  - user
  - prisma
  - match
  - stat
    main.ts
    app.module.ts
- controllers
- routes
- prisma
- node-modules

  \* Files:

- .env (generate your own)
- .gitignore
- package.json
- package-lock.json
- (this) readme.md

#### Database should contain 3 tables:

- Users(<ins>username</ins>, email, password)
- Stats(<ins>username</ins>, victories, defeats, draws)
- Matches(<ins>id</ins>, username1->Users, username2->Users, score1, score2, createdAt)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash

# e2e tests
$ npm run test:e2e

$ npm run test:cov
```
